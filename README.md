🧠 About the Project

AI Project Builder is a multi-agent AI system that transforms simple user ideas into complete machine learning projects.

Just type:

«“Build a fake news detection system using NLP”»

And get:

- 📌 Structured Project Plan
- 💻 Complete Python Code
- ✅ Improved & Optimized Code
- 📖 Beginner-Friendly Explanation

---

🤖 Multi-Agent Architecture

This project simulates real-world AI systems using specialized agents:

- 🧠 Planner Agent → Designs project roadmap
- 💻 Code Generator Agent → Writes ML code
- 🔍 Reviewer Agent → Optimizes & improves code
- 📖 Explainer Agent → Teaches concepts simply

---

⚙️ Tech Stack

- 🐍 Python
- 🎨 Streamlit (UI)
- ⚡ Groq API (LLM inference)
- 📊 Pandas, Scikit-learn, Matplotlib
